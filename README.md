# Admin Panel with Passport Authentication

A secure **Admin Panel** built with **Node.js, Express.js, MongoDB, EJS, Passport.js, and Multer**. This project features user authentication, session management, image uploads, and complete Blog CRUD operations.

## 🚀 Features

* User Registration with Password Hashing (bcrypt)
* Secure Login & Logout using Passport.js (Local Strategy)
* Express Session Authentication
* Protected Routes
* Blog Management (Create, Read, Update, Delete)
* Image Upload with Multer
* MongoDB Database Integration using Mongoose
* EJS Templating Engine
* Cookie & Session Management
* Responsive Admin Dashboard
* Error Handling Pages

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* Passport.js
* Passport Local Strategy
* Express Session
* bcrypt
* Multer
* EJS
* Cookie
